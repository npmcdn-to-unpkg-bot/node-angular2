# node-angular2
My Test NodeJS with Angular2 project, to better understand the two working together.

# About the boilerplate
<ul>
    <li>boilerplate 2.0</li>
    <li>gulp integrated</li>
    <li>browser-sync integrated</li>
</ul>

# How to use
<ul>
    <li>clone repro</li>
    <li>npm install</li>
    <li>npm run gulp</li>
    <li>npm start</li>
</ul>
